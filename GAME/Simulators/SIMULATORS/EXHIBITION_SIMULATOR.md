# 特展模拟器 | Exhibition Simulator

> **状态**: 🔄 规划中
> **类型:** 策展模拟 / 空间叙事

---

## 一、核心理念

特展模拟器让玩家扮演**策展人**角色，策划并执行一个完整的艺术展览。

**与艺术家模拟器的区别**:
- 艺术家模拟器：个人创作生涯
- 特展模拟器：策展权力与行业生态

**核心问题**: 展览作为"话语权"的载体，策展人如何定义"什么是艺术"？

---

## 二、核心玩法

### 2.1 展览策划阶段

| 步骤 | 内容 |
|------|------|
| 主题确定 | 选择展览主题、学术叙事、核心命题 |
| 空间设计 | 规划展览动线、空间分割、展品布局 |
| 作品选择 | 从艺术家库选择作品，或委托创作 |
| 协调借展 | 与美术馆、收藏家、艺术家谈判 |
| 预算分配 | 场地、运输、保险、出版、宣传 |

### 2.2 展览运营阶段

| 步骤 | 内容 |
|------|------|
| 开幕筹备 | 邀请VIP、媒体、艺术家 |
| 公众导览 | 培训导览员、编写导览词 |
| 展期活动 | 讲座、工作坊、表演安排 |
| 观众反馈 | 收集数据、分析反馈 |

### 2.3 展览评估阶段

| 步骤 | 内容 |
|------|------|
| 数据复盘 | 观众数量、媒体曝光、社交讨论 |
| 财务结算 | 收支平衡、赞助商反馈 |
| 学术评价 | 评论文章、奖项提名 |
| 权力博弈 | 各方利益博弈、后续合作 |

## 四、特殊事件示例

### 4.1 现实级

**艺术家临时撤展**
> "抱歉，由于个人原因，我决定撤回参展作品。"

触发条件：展览开幕前30天
效果：学术资本-20，焦虑值+30

### 4.2 讽刺级

**策展人身份危机**
> "一个从未做过艺术的策展人，有什么资格定义艺术？"

某社交媒体用户在开幕式上提问："你做过艺术吗？你有什么资格策展？你懂艺术吗？"

触发条件：争议指数>50
效果：争议指数+30，公众关注+20

### 4.3 荒诞级

**作品在开幕当天消失**
> "监控显示，作品在凌晨3:17分自己走进了仓库。"

一件互动装置在开幕当天消失，现场只留下一张纸条："我感受到了策展人的不真诚，所以我选择离开。"

触发条件：任意
效果：范式值+50，争议指数+40

## 五、多人协作（扩展）

### 5.1 角色系统

| 角色 | 功能 |
|------|------|
| 主策展人 | 整体把控、最终决策 |
| 助理策展人 | 执行落实 |
| 空间设计师 | 视觉规划 |
| 媒体联络 | 公共关系 |
| 收藏家代表 | 资金支持 |

### 5.2 博弈机制

- 不同角色有不同利益诉求
- 策展人需要在各方之间平衡
- 决策会影响最终展览呈现

  id: "exhibition_001",
