# 艺术博览会模拟器 | Art Fair Simulator

> **状态**: 🔄 规划中
> **类型:** 商业生态模拟 / 多方博弈

---

## 一、核心理念

艺术博览会模拟器让玩家体验**大型艺博会**的整体运营——多展位管理、交易博弈、行业趋势。

**与其他模拟器的区别**:
- 艺术家模拟器：个人创作
- 特展模拟器：策展单一展览
- 拍卖模拟器：单件作品交易
- 艺博模拟器：大规模商业生态

**核心问题**: 当艺术成为商品，艺博会如何平衡商业与学术？

## 二、核心玩法

### 2.1 展会筹备

| 步骤 | 内容 |
|------|------|
| 展位申请 | 争取热门展位 |
| 空间设计 | 展位装修、视觉 |
| 作品选择 | 带什么作品、定价多少 |
| 销售策略 | 目标藏家、VIP邀请 |
| 团队分工 | 销售/接待/讲解 |

### 2.2 展会期间

| 步骤 | 内容 |
|------|------|
| VIP预览 | 首日面向VIP的销售 |
| 公众日 | 面向大众的展示 |
| 同行交流 | 与其他画廊信息互换 |
| 媒体应对 | 接受采访、曝光 |
| 即时调整 | 根据销售情况调价 |

### 2.3 展会结束

| 步骤 | 内容 |
|------|------|
| 销售结算 | 统计成交、佣金 |
| 客户跟进 | 维护藏家关系 |
| 经验复盘 | 什么卖得好、什么没人要 |
| 行业评价 | 同行口碑、趋势判断 |

## 四、多方博弈

### 4.1 核心角色

| 角色 | 诉求 | 博弈对象 |
|------|------|----------|
| 画廊主 | 销售最大化 | 藏家、艺术家 |
| 藏家 | 作品稀缺性 | 其他藏家 |
| 策展人 | 学术话语权 | 画廊主 |
| 艺博方 | 展会影响力 | 所有参与者 |
| 艺术家 | 市场认可 | 画廊主 |

### 4.2 博弈场景

**定价博弈**:
- 高价 → 高利润但可能卖不掉
- 低价 → 销售快但可能亏

**信息博弈**:
- 藏家互相不知道对方出什么价
- 画廊主不知道藏家的心理价位

**时间博弈**:
- VIP预览抢第一时间
- 公众日捡漏

## 五、特殊事件示例

### 5.1 现实级

**隔壁展位卖掉了**
> "隔壁展位刚卖掉一幅200万的！"

开展第一小时，隔壁展位传来成交消息，刺激你的团队加速销售。

触发条件：展会开始后1小时
效果：焦虑值+10，销售速度+20%

### 5.2 讽刺级

**藏家要求"艺术顾问费"**
> "我可以帮你卖，但你要给我成交价15%的顾问费。"

一位"资深藏家"表示可以介绍客户，但需要额外抽成——这在艺博会是公开的秘密。

触发条件：销售额<50万
效果：销售成本+15%，讽刺值+20

### 5.3 荒诞级

**作品被观众吃掉了**
> "那位先生把展位上的糖果作品吃掉了。"

一位观众把你展位上的"可食用艺术装置"当作自助餐吃了，还问你要不要报警。

触发条件：展位有可食用作品
效果：行业知名度+30，销售额-10%
