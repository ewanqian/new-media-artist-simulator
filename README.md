## 引言

《新媒体艺术家模拟器》是一件基于人工智能技术创作的超文本交互当代艺术作品，由艺术家钱誉文（EWAN CT）于2024年发起创作，最初为杭州中心美术馆「巴别瓶：人工智能时代的语用、创造与生活形式」展览的委托装置作品，2026年完成主干版的全面升维迭代。

作品以游戏化的生涯模拟为载体，以黑色幽默的文本解构为核心，以AI时代艺术家的主体性危机为核心命题，构建了一个兼具观念性、工具性、交互性与成长性的艺术系统。

---

## 一、时代语境与核心命题

### 1.1 媒介范式的转移：从「新媒体艺术」到「泛在媒体艺术」

2026年，生成式人工智能技术对创意领域的改造已进入深水区，数字媒介完成了从「先锋创作工具」到「泛在生产资料」的根本性转变，新媒体艺术的定义边界正在被彻底重构。

上世纪末至本世纪前二十年，新媒体艺术的先锋性始终与「媒介实验性」深度绑定：艺术家通过掌握创意编程、交互装置开发、沉浸式影像技术等专属技能，突破传统艺术的媒介边界，构建新的艺术语言。而在生成式AI普及的当下，代码编写、视觉生成、交互设计、方案撰写等曾经的专业门槛被降低至近乎为零，数字媒介不再是少数创作者的专属实验场，而是所有创作者都可随手使用的基础工具。

媒介范式的转移，直接引发了当代艺术场域的系统性震荡：当技术不再是壁垒，当AI可以完成大部分创作执行环节，新媒体艺术家的身份认同、核心价值与创作合法性，都面临着前所未有的挑战。

### 1.2 艺术场域的双重博弈：符号资本与经济资本的撕裂

在媒介革命的同时，中国当代艺术场域内部的二元博弈正在被持续放大。根据布尔迪厄的艺术场域理论，文化生产场域始终存在「有限生产场域」与「大规模生产场域」的对立：前者以学术合法性、符号资本为核心评价准则，追求艺术的自主性与先锋性；后者以市场价值、经济资本为核心逻辑，以商业回报为核心目标。

在学术体系内，新媒体艺术已成为艺术院校扩招、留学申请的核心赛道，大量创作者为了积累符号资本，陷入了「黑话内卷」的话语游戏。在商业市场中，新媒体艺术则被简化为「沉浸式打卡装置」「AI视觉特效」，文旅项目、商业空间、品牌活动对新媒体内容的需求空前旺盛。

### 1.3 核心命题：AI自动化时代的创作主体性危机

当AI可以完成从创意构思、文本撰写、视觉生成、代码编写到方案输出的全流程工作，创作者正在陷入深刻的存在性虚无：我们究竟是创作的主体，还是AI生成内容的筛选者？
