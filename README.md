## 🎨 关于作品

《新媒体艺术家模拟器》是一个具有讽刺意味与反思精神的交互式超文本作品，邀请你扮演一位新媒体艺术家，在理想与现实的夹缝中，在AI重构创作边界的时代，体验艺术创作的荒诞与真实。

作品最初由钱誉文 EWAN CT 创作于2024年，作为「巴别瓶：人工智能时代的语用、创造与生活形式」展览的特别委托作品，于杭州中心美术馆CAM展出。2026年，作品升级为主干版本，从一个生涯模拟游戏，进化为一个完整的新媒体艺术思想实验场。

### 核心概念

在人工智能彻底消解创作门槛的后数字时代，艺术家的身份认同、创作价值、主体性，正面临前所未有的挑战。本作品用算法逻辑模拟艺术家的职业生涯，以游戏化的方式呈现艺术体制的运作机制，质询当代艺术生产中的权力结构与价值判断。

作品本身使用AI辅助开发，形成了完整的自反性创作实践：用人工智能讨论人工智能时代的艺术创作，用模拟的方式揭示现实的荒诞。

## 🎮 核心体验

### 八维数值系统

用八个维度构建艺术家的完整生存状态：

**专业维度**：理论深度、学术资本、范式突破、研究经费  
**生存维度**：焦虑指数、讽刺值、现实感、幽默度

### 四大行动类别

- **🏛️ 研究场域** - 探索各种前卫实验空间
- **📚 专业准备** - 进行技能学习与理论研究
- **🎨 项目研究** - 启动新媒体艺术创作项目
- **💰 筹措经费** - 通过各种方式维持艺术实践

### 三层文本结构

每一个选项都包含：
- 正式的艺术话语
- 通俗的现实翻译
- 尖锐的讽刺评论

### 六大艺术流派（2026版）

- **媒介反思派**：用科技但克制，反思媒介本身
- **生成混沌派**：纯AI生成，探索生成艺术边界
- **空间沉浸派**：XR、空间计算、沉浸式文旅
- **代码原生派**：坚持纯手工创意编程，拒绝AI
- **社会介入派**：用新媒体技术做公共艺术
- **跨媒介融合派**：数字媒介与传统实体媒介融合

## 🎭 理论定位

作品建立在三大当代艺术理论的基础上：

### 布尔迪厄「艺术场域」理论

八维数值系统是对艺术场域理论的游戏化转译——艺术场域是一个充满斗争的空间，艺术家在「有限生产场域」（学术、纯粹艺术）和「大规模生产场域」（市场、商业）之间不断博弈。

### 福柯「话语权力」理论

三层文本结构是对话语权力理论的实践——当代艺术的"专业话语"，本质上是一种权力体系。作品用三层结构拆解了艺术话语的权力泡沫。

### 后数字时代的媒介艺术理论

作品回应了后数字时代、AI爆发后的艺术危机——当AI彻底消解了数字技术的创作门槛，新媒体艺术不再是"用新技术做艺术"，而是"对新技术的批判性使用"。
