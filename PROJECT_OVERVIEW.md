## 一、项目愿景

《新媒体艺术家模拟器》是一个**AI时代艺术创作的思想实验场**，同时具备以下多重属性：

1. **交互式当代艺术作品** - 基于游戏化生涯模拟的超文本艺术实践
2. **艺术场域认知教育工具** - 帮助公众理解新媒体艺术行业的生态与博弈
3. **创作者自我批判与反思工具** - 为职业艺术家提供多视角的自我审视框架
4. **学术研究平台** - 探索AI时代创作主体性的理论与实践

**核心命题**: 在AI自动化的浪潮中，艺术家的主体性到底是什么？

## 二、系统架构

### 2.1 核心模拟器生态

| 模拟器 | 状态 | 描述 |
|--------|------|------|
| 艺术家生涯模拟器 | ✅ 已实现 | 核心体验模块，八维数值系统，三层文本结构 |
| 特展模拟器 | 🔄 规划中 | 模拟策展、空间设计、作品选择、观众互动 |
| 艺术品拍卖模拟器 | 🔄 规划中 | 模拟艺术品市场、竞价机制、收藏家行为 |
| 艺术博览会模拟器 | 🔄 规划中 | 模拟大型艺博会、多展位管理、交易博弈 |

### 2.2 文本系统

**多调性文本架构** - 解决"可控性"与"模糊性"的平衡：

| 调性 | 等级 | 特征 | 用途 |
|------|------|------|------|
| 严肃调 | 现实级 | 基于真实艺术生态数据 | 学术研究、训练数据、理性决策 |
| 讽刺调 | 讽刺级 | 黑色幽默、体制批判 | 核心特色、批判性、玩家共鸣 |
| 荒诞调 | 荒诞级 | 离谱但有理论基础 | 娱乐性、游戏趣味、意外体验 |
| 积极调 | 赋能级 | 鼓励、引导、建设性 | 共情支持、发展建议 |

## 四、理论锚点

### 4.1 核心理论支柱

1. **布尔迪厄艺术场域理论** - 游戏化转译八维数值系统
2. **福柯话语权力理论** - 三层文本结构实现话语祛魅
3. **后数字时代媒介理论** - 回应AI时代的媒介变革
4. **AI时代主体性批判** - 探索创作者的核心价值

### 4.2 核心理论概念

- **符号资本与经济资本的撕裂** - 学术场域vs商业市场的二元博弈
- **话语泡沫与创作空洞化** - 理论内卷带来的异化
- **AI自动化与创作主体性危机** - 技术vs人的核心命题
- **艺术场域的颠倒逻辑** - "输者赢"的先锋性准则
