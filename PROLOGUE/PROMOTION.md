## 朋友圈文案

### 版本1：简洁版

```
🎮 做了一个「新媒体艺术家模拟器」的序章Demo

你是一个刚踏入艺术圈的年轻人。
可以选：美院毕业生/AI转行/海归/自由职业...

在艺术区租工作室、见策展人、做作品、办展览。
每个选择都会记录，生成你的「艺术生涯轨迹」。

玩的越多，后面玩家遇到的NPC可能就是你的化身。
你创作的作品，会成为别人游戏里的「传说」。

有点《赛博朋克2077》选择那个味儿了，但做的是艺术圈生态。

有兴趣的可以来玩内测🔗
```

### 版本2：详细版

```
🎮 做了一个很「真」的艺术家模拟器序章

你扮演一个2026年的新媒体艺术家。
从8种身份开始：美院毕业生、AI工程师转行、海归、自由职业...

在虚构的「榕树湾艺术区」租工作室、跑展览、见策展人、跟收藏家打交道。
做装置、办展览、应对各种鸡毛蒜皮的破事。

【最特别的是】
你的每个选择都会被记录。
- 你是倾向学术还是商业？
- 你更看重钱还是艺术理想？
- 你遇到困难会硬刚还是妥协？

玩到一定程度，你的角色会变成「NPC」——
后面来玩的玩家可能会遇到「你」，跟你对话。

你做过的作品，也会成为这个虚拟艺术世界的「传说」。

玩的人越多，这个艺术生态就越丰富。
每个人都是构建者。

有点像是艺术圈的《赛博朋克2077》+《黑魂》入侵系统？

内测入口 🔗
```

### 版本3：吊胃口版

```
🤔 有没有一个游戏，能让你体验当艺术家？

不是那种打怪升级的RPG。
是那种——

你刚毕业，没钱，在艺术区租了个破工作室。
隔壁有个毒舌老炮儿天天吐槽你。
有个策展人问你「你想回答什么问题」。
你做了一个东西，展出去，有人买，有人骂。

然后你发现——
之前玩过的人，他们的角色变成了NPC。
你做的作品，变成了别人的任务道具。

这个「艺术世界」，是所有玩家一起构建的。

我们叫它「新媒体艺术家模拟器」的序章。

想玩的🔗
```

## 核心卖点总结（发给团队内部）

```
1. 8种身份起手
   美院毕业生/AI工程师转行/海归/自由职业/传统画家转型/独立艺术家/画廊从业者/程序员跨界

2. 真实的艺术生态
   虚构的「榕树湾艺术区」，但基于真实艺术圈逻辑
   租工作室、见策展人、跑展览、跟收藏家博弈

3. 选择即塑造
   每个选择记录下来，形成你的「艺术人格」
   学术派还是商业派？理想主义还是生存优先？

4. 玩家共建世界
   玩到一定程度，你变成NPC
   后面玩家可能遇到「你」，跟你对话
   你的作品成为别人的任务道具

5. 15-60分钟体验
   快节奏15分钟了解系统
   慢节奏60分钟深度体验

目标：让每个人都能在游戏里体验一次「艺术生涯」，同时构建一个真实的虚拟艺术生态。
```
