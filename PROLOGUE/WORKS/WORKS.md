  # 基础信息
  id: "WORK_UUID"
  title: "作品名称"
  artist: "作者ID"
  created_at: "创作时间"

    faction_tags: ["派系标签"]

  # 描述

  # 展览历史
  exhibition_history:
    - exhibition名称"
      date: "展览: "展览日期"

  # 评价数据
  reception:

  # 数值影响
  impact:
    theory: +5
    academic: +10

    funding: +3

## 作品创作流程

### 步骤

1. **概念构思**
   - 确定创作主题
   - 选择派系方向
   - 设定目标受众

2. **技术实现**
   - 选择媒介
   - 确定技术栈
   - 制作原型

3. **完成创作**
   - 完善作品细节
   - 撰写艺术家自述
   - 设定作品价格

4. **发布展示**
   - 选择展示方式
   - 参加展览/销售
   - 收集反馈
