# 工作坊_国美艺术工作坊

## 工作坊简介

中国美术学院官方艺术工作坊，面向社会艺术爱好者及专业创作者

## 课程体系

### 国画书法
- 写意山水入门
- 工笔花鸟
- 书法基础
- 课时：每周2次，持续8周

### 油画
- 油画材料技法
- 风景写生
- 人物创作
- 课时：每周2次，持续8周

### 版画
- 木版版画
- 铜版版画
- 版画创作
- 课时：每周2次，持续10周

### 当代艺术
- 观念艺术工作坊
- 装置艺术入门
- 行为艺术实践
- 课时：每周2次，持续6周

### 数字艺术
- 3D建模基础
- 数字雕塑
- 虚拟现实艺术
- 课时：每周2次，持续8周
