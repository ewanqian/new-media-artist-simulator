# 小红书_东京新媒体艺术

## 🔵东京新媒体艺术｜科技艺术圣地巡礼

**收藏：东京新媒体艺术**

 新媒体艺术/交互设计/装置艺术的宝子必看！

🎯 **必去机构**

1. **TeamLab Borderless/Planets**
   - 东京最火的新媒体艺术体验馆
   - 沉浸式光影装置，超级出片
   - 建议工作日去，周末人超多

2. **21_21 Design Sight**
   - 安藤忠雄设计，三宅一生主导
   - 经常有高质量设计/媒体艺术展
   - 票有点贵但值得

3. **日本科学未来馆**
   - 交互装置很多
   - 可以看到最前沿的科技艺术结合

4. **东京现代美术馆**
   - 收藏丰富，包括草间弥生、奈良美智
   - 也有新媒体艺术常设展

📍 **下北泽** - 艺术家聚集地
- 很多独立画廊和艺术空间
- 周末会有艺术市集

📍 **涩谷** - 科技×艺术
- 十字路口的LED大屏是天然新媒体艺术
- 很多科技公司艺术部门

💡 **学习资源**
- 东京艺术大学媒体艺术科
- 庆应KMD媒体设计
- 多摩美术大学情报设计
