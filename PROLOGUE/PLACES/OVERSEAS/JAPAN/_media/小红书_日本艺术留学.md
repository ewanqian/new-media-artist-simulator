# 小红书_日本艺术留学

## 🎌日本艺术留学｜超全择校指南

**收藏：艺术留学日本**

 分享给想要去日本学艺术的宝子们！

🎨 **推荐院校**

| 学校 | 特色 | 学费/年 |
|------|------|---------|
| 东京艺术大学 | 日本最高艺术学府 | ~30万日元 |
| 武蔵野美术大学 | 设计类最强 | ~80-120万日元 |
| 多摩美术大学 | 平面/产品强 | ~80-120万日元 |
| 庆应义塾大学KMD | 媒体设计/创新 | ~100万日元 |
| 东京大学GS | 综合类艺术研究 | ~50万日元 |

📝 **申请要点**

1. **语言**：日语N2是底线，N1更稳
2. **作品集**：日本风格偏概念性，重视制作过程
3. **研究计划书**：必须认真写，教授很看重
4. **考试**：需要去日本参加现场考试（素描/小论文/面试）

💰 **开销参考**
- 学费：30-120万日元/年
- 房租：东京5-8万日元/月
- 生活费：8-15万日元/月
- 兼职：时薪1000-1500日元

🎯 **过来人建议**
- 语言学校读半年-1年再考修士更稳
- 京都艺术大学、金泽美术工艺大学也很不错
- 国立大学学费便宜但难考
- 作品集可以找日本老师辅导，更对口味
