# 项目路线图 | Project Roadmap

> **版本**: 2026主干版
> **更新时间**: 2026年2月

---

## 项目愿景

《新媒体艺术家模拟器》是一件AI时代的交互式艺术作品，通过游戏化生涯模拟，让玩家体验新媒体艺术家的真实生存状态，思考艺术主体性的本质。

---

## 技术规划

### 技术栈

| 模块 | 技术 | 状态 |
|------|------|------|
| 前端 | React/Vue | 现有 |
| 游戏引擎 | 原生JS | 现有 |
| 后端 | Node.js | 现有 |
| 数据库 | JSON | 现有 |
| AI对话 | OpenAI API | 规划 |
| 线上平台 | 云服务 | 规划 |

### 架构升级

1. **单体 → 模块化**: 将游戏拆分为独立模块
2. **本地 → 云端**: 支持线上数据和多人
3. **固定 → AI**: NPC对话智能化
