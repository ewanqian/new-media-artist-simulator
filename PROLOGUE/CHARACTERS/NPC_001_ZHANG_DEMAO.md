**早期经历**：
- 1990-1995年：在浙省各地写生，创作乡土油画
- 1996-2000年：北漂，在宋庄做当代艺术
- 2001-2010年：回到南方，开始做装置和影像
- 2010-至今：定居榕州市，做独立艺术家

**代表作品**：
- 《水泥森林》系列（油画，1998）
- 《废墟上的 dance》（装置，2005）
- 《二维码时代的渔民》（影像，2015）

**艺术风格**：
- 早期：乡土现实主义
- 中期：社会批判装置
- 近期：关注技术与人性的关系

## 相遇方式

### 场景1：共享空间B区201室

**时间**：周一上午10点

**触发条件**：
- 玩家进入共享空间整理工作室

**对话内容**：
```
[你正在整理从淘宝买的便宜画材，突然有人敲门]

张德茂："哎，新来的？我老张，住隔壁203。听说你是刚毕业的？"

[他不由分说地走进来，环顾四周]

张德茂："就这破地方，比我当年在宋庄的条件还差。
当年我们在村里租个院子，一个月才200块。
你们这代人，什么都没经历过，就会花钱。"

你："...张叔，我这刚开始，资金有限..."
张德茂："资金有限？我刚毕业的时候，连画材都买不起！
你知道我用什么画画吗？，用油漆！
便宜的油漆，画在水泥地上，风一吹就没了。
艺术这个东西，不是用钱堆的，是用命堆的！"
```

### 场景2：榕江边大排档

**时间**：周末晚上8点

**触发条件**：
- 玩家在榕江边散步

**对话内容**：
```
[你正在江边抽烟，突然听到有人打电话]

张德茂（对着手机）："李总，我说了多少遍了，我的作品不打折！
当年我在宋庄的时候，有人要买我的画，我都没卖！
艺术是能用钱衡量的吗？
好好好，见面聊，不喝白酒啊，我胃不好..."

[他挂掉电话，看到你]

张德茂："哎，刚毕业的吧？来，抽一根。"
[他递给你一根烟]
张德茂："刚才那个，画廊老板，想让我降价。
开玩笑，我画了30年，凭什么给他打折？
你知道他转手卖多少钱吗？三倍！
这就是艺术市场，狗日的。"
```

## 语录集

### 毒舌语录

1. 关于AI：
> "AI做艺术？搞笑！那叫打印，不叫画画！
你见过打印机喊累的吗？艺术是要痛的，AI懂什么叫痛？"

2. 关于年轻艺术家：
> "现在的年轻人，一毕业就想着卖作品。
我们那时候，想的是怎么活下去。
你们啊，太幸福了，所以做不出好东西。"

3. 关于策展人：
> "策展人？那帮孙子懂什么艺术！
他们就知道概念，概念，概念。
画一幅画出来，他们能说出8000字分析报告，你信不信？"

4. 关于收藏家：
> "收藏家？百分之九十都是投资客。
真懂艺术的，没有几个。
他们买的是名字，不是作品。"

5. 关于自己：
> "我画了30年，一幅画没卖出去，
但我还活着，还能画，这就够了。"

1. **真实性**：老炮张的设定基于真实的"老艺术家"形象
