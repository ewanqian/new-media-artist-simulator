## 相遇方式

### 场景1：南岭艺术学院讲座

**时间**：周三下午2点

**触发条件**：
- 玩家参加"策展理论与实践"讲座

**对话内容**：
```
[讲座结束，你正在收拾东西]

王明远："这位同学，你是哪个专业的？"

你："老师好，我是刚毕业的，想做新媒体艺术..."

王明远："新媒体艺术？嗯，现在年轻人都在做。
你知道新媒体艺术的核心是什么吗？"

你："...技术？"

王明远："错，是问题。
技术只是手段，问题才是核心。
你要回答什么问题？
AI来了，艺术家怎么办？
技术发展，人还剩下什么？
你想清楚了，再来做艺术。"
```

### 场景2：榕树湾艺术中心

**时间**：周六下午4点

**触发条件**：
- 玩家参观艺术中心展览

**对话内容**：
```
[你正在看展，突然有人走到旁边]

王明远："你也对这个展览感兴趣？"

你："老师好！这个作品很有意思..."

王明远："有意思？你说说，什么意思？"

你："就是...那种...说不清的感觉..."

王明远："说不清就对了。
当代艺术，就是让你说不清。
说不清，才能继续说。
如果什么东西都说清了，艺术就死了。"
```

## 语录集

### 学术语录

1. 关于策展：
> "策展不是策展人的自我表达，是让艺术家和观众对话。
策展人是桥梁，不是主角。"

2. 关于当代艺术：
> "当代艺术不是艺术，是问题。
你做作品，是为了回答问题。
没有问题，就没有当代艺术。"

3. 关于AI：
> "AI时代，最不值钱的是技术，最值钱的是问题。
因为AI可以帮你做，但AI不知道该做什么。"

4. 关于学术：
> "学术不是往上爬的梯子，是往下扎的根。
你能扎多深，决定你能走多远。"

5. 关于学生：
> "我培养学生，不是让他们成为策展人，是让他们成为思考者。
策展只是思考的出口，不是入口。"
