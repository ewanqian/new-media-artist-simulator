# 随机事件系统

本文件定义了游戏中随机触发的意外事件，这些事件会增加游戏的不可预测性和趣味性。

---

## 1. 灵感爆发 (Inspiration Surge)

### 1.1 突发灵感
- **事件ID**: AC_INSPIRE_001
- **名称**: 突发灵感！
- **类型**: 正面随机事件
- **触发条件**: 随机概率触发，与玩家当前活动无关
- **效果描述**: 在日常生活中突然获得创作灵感
- **奖励**:
  - 灵感 +30~50
  - 创意点数 +1~2
- **消耗**: 无
- **备注**: 可触发创作冲动事件

### 1.2 梦中启示
- **事件ID**: AC_INSPIRE_002
- **名称**: 梦中的创作
- **类型**: 正面随机事件
- **触发条件**: 睡眠后随机触发
- **效果描述**: 做了一个奇怪的梦，获得了惊人的创作灵感
- **奖励**:
  - 灵感 +40~60
  - 特殊作品创意 +1
- **消耗**: 无
- **备注**: 可能创作出独特作品

### 1.3 跨界启发
- **事件ID**: AC_INSPIRE_003
- **名称**: 跨界的火花
- **类型**: 正面随机事件
- **触发条件**: 接触其他领域后概率触发
- **效果描述**: 从音乐/科学/自然等领域获得创作启发
- **奖励**:
  - 灵感 +25~40
  - 跨学科创意 +1
- **消耗**: 无

## 2. 创作瓶颈 (Creative Block)

### 2.1 灵感枯竭
- **事件ID**: AC_BLOCK_001
- **名称**: 创作瓶颈
- **类型**: 负面随机事件
- **触发条件**: 连续创作后概率触发
- **效果描述**: 突然不知道要创作什么，陷入创作困境
- **奖励**: 无
- **消耗**:
  - 灵感 -30~50
  - 创作速度 -20% (持续3-7天)
- **解决方案**: 
  - 参加放松活动
  - 接触新事物
  - 等待自然恢复

### 2.2 完美主义陷阱
- **事件ID**: AC_BLOCK_002
- **名称**: 无法满足的完美主义
- **类型**: 负面随机事件
- **触发条件**: 高名望玩家概率触发
- **效果描述**: 对作品过度苛求，无法完成任何作品
- **奖励**: 无
- **消耗**:
  - 创作时间 +50%
  - 作品完成度下降
- **解决方案**: 
  - 降低自我要求
  - 接受不完美
  - 寻求他人鼓励

### 2.3 技术障碍
- **事件ID**: AC_BLOCK_003
- **名称**: 技术难题
- **类型**: 负面随机事件
- **触发条件**: 使用新技术创作时概率触发
- **效果描述**: 遇到无法解决的技术问题
- **奖励**: 无
- **消耗**:
  - 进度暂停 1-3天
  - 可能需要学习新技术
- **解决方案**: 
  - 参加技术工作坊
  - 求助技术专家
  - 更换创作方向

## 3. 作品损坏 (Work Damage)

### 3.1 意外损毁
- **事件ID**: AC_DAMAGE_001
- **名称**: 作品意外损毁
- **类型**: 负面随机事件
- **触发条件**: 作品保存时概率触发
- **效果描述**: 由于意外（火灾/水淹/碰撞等）作品受损
- **奖励**: 无
- **消耗**:
  - 失去 1件作品
  - 金钱 -¥5,000~20,000 (修复或重建)
  - 名望 -5~15 (如果需要取消展览)
- **备注**: 有保险可减少损失

### 3.2 运输损坏
- **事件ID**: AC_DAMAGE_002
- **名称**: 运输中的意外
- **类型**: 负面随机事件
- **触发条件**: 作品运输时概率触发
- **效果描述**: 作品在运输过程中受损
- **奖励**: 无
- **消耗**:
  - 失去 1件作品
  - 金钱 -¥10,000~50,000
  - 展览可能受影响
- **备注**: 购买保险可获赔偿

### 3.3 数字作品丢失
- **事件ID**: AC_DAMAGE_003
- **名称**: 数字文件丢失
- **类型**: 负面随机事件
- **触发条件**: 数字创作时概率触发
- **效果描述**: 硬盘损坏/文件误删等导致作品丢失
- **奖励**: 无
- **消耗**:
  - 失去数字作品文件
  - 时间 -1~3天 (重建)
- **备注**: 定期备份可避免

## 4. 遇到贵人 (Meeting Benefactor)

### 4.1 收藏家青睐
- **事件ID**: AC_BENEFACTOR_001
- **名称**: 遇到伯乐
- **类型**: 正面随机事件
- **触发条件**: 展览或社交活动中概率触发
- **效果描述**: 被知名收藏家看中
- **奖励**:
  - 关系 +30~50 (收藏家)
  - 名望 +20~30
  - 金钱 +¥10,000~50,000 (作品收购)
- **消耗**: 无
- **备注**: 长期合作机会

### 4.2 画廊签约
- **事件ID**: AC_BENEFACTOR_002
- **名称**: 画廊邀约
- **类型**: 正面随机事件
- **触发条件**: 名望达到一定程度后触发
- **效果描述**: 收到画廊的合作邀请
- **奖励**:
  - 关系 +40~60 (画廊)
  - 名望 +25~40
  - 稳定收入来源
- **消耗**: 签订独家代理合约

### 4.3 媒体曝光
- **事件ID**: AC_BENEFACTOR_003
- **名称**: 媒体关注
- **类型**: 正面随机事件
- **触发条件**: 作品有特色时概率触发
- **效果描述**: 受到媒体关注并报道
- **奖励**:
  - 媒体关系 +25~40
  - 名望 +15~25
  - 公众曝光 +20~30
- **消耗**: 接受采访/拍摄

### 4.4 投资人出现
- **事件ID**: AC_BENEFACTOR_004
- **名称**: 天使投资
- **类型**: 正面随机事件
- **触发条件**: 有商业潜力时概率触发
- **效果描述**: 被艺术投资人看中
- **奖励**:
  - 金钱 +¥50,000~200,000 (投资)
  - 商业关系 +20~30
  - 发展机会
- **消耗**: 股权/分成协议

## 5. 丢失材料 (Lost Materials)

### 5.1 采购丢失
- **事件ID**: AC_LOST_001
- **名称**: 材料采购丢失
- **类型**: 负面随机事件
- **触发条件**: 购买材料后概率触发
- **效果描述**: 购买的创作材料在运输中丢失
- **奖励**: 无
- **消耗**:
  - 金钱 -¥2,000~10,000
  - 创作延迟 1-3天
- **备注**: 可联系卖家补发

### 5.2 库存材料损坏
- **事件ID**: AC_LOST_002
- **名称**: 库存受损
- **类型**: 负面随机事件
- **触发条件**: 库存材料保存时概率触发
- **效果描述**: 保存的材料过期/损坏
- **奖励**: 无
- **消耗**:
  - 失去部分材料
  - 金钱 -¥1,000~5,000
- **备注**: 妥善保存可避免

### 5.3 灵感笔记本丢失
- **事件ID**: AC_LOST_003
- **名称**: 灵感笔记丢失
- **类型**: 负面随机事件
- **触发条件**: 有记录习惯时概率触发
- **效果描述**: 记录灵感的笔记本丢失
- **奖励**: 无
- **消耗**:
  - 失去部分创意记录
  - 灵感 -10~20
- **备注**: 云备份可避免

## 6. 其他随机事件

### 6.1 作品获奖
- **事件ID**: AC_AWARD_001
- **名称**: 意外获奖
- **类型**: 正面随机事件
- **触发条件**: 参赛作品后随机触发
- **效果描述**: 作品意外获得奖项
- **奖励**:
  - 名望 +30~60
  - 金钱 +¥10,000~50,000
  - 关系 +20~30
- **消耗**: 领奖时间

### 6.2 抄袭风波
- **事件ID**: AC_CONTROVERSY_001
- **名称**: 抄袭指控
- **类型**: 负面随机事件
- **触发条件**: 高名望时概率触发
- **效果描述**: 被指控作品抄袭
- **奖励**: 无 (如澄清则名望+)
- **消耗**:
  - 名望 -20~40 (如属实)
  - 时间用于澄清
  - 金钱 -¥5,000~20,000 (法律/公关)
- **解决方案**: 
  - 证明原创性
  - 道歉并赔偿

### 6.3 健康问题
- **事件ID**: AC_HEALTH_001
- **名称**: 创作过度
- **类型**: 负面随机事件
- **触发条件**: 连续高强度工作后触发
- **效果描述**: 因过度创作导致健康问题
- **奖励**: 无
- **消耗**:
  - 体力 -30~50
  - 创作暂停 3-7天
- **解决方案**: 
  - 休息
  - 调整作息

### 6.4 竞争对手
- **事件ID**: AC_RIVAL_001
- **名称**: 新锐竞争对手
- **类型**: 中性随机事件
- **触发条件**: 名望较高时触发
- **效果描述**: 出现风格相似的新艺术家
- **效果**: 
  - 竞争压力
  - 市场可能被分占
- **应对**: 
  - 差异化发展
  - 保持创作质量

### 6.5 意外合作机会
- **事件ID**: AC_COLLAB_001
- **名称**: 跨界合作邀请
- **类型**: 正面随机事件
- **触发条件**: 社交活动中概率触发
- **效果描述**: 收到跨界合作邀请（品牌/科技/电影等）
- **奖励**:
  - 关系 +25~40
  - 名望 +15~25
  - 金钱 +¥20,000~100,000
  - 新技能/领域了解
- **消耗**: 时间投入

## 事件触发概率

- **正面事件**: 总体概率约 30%
- **负面事件**: 总体概率约 25%
- **中性事件**: 总体概率约 15%
- **无事件**: 约 30%

### 影响概率因素
- 名望越高，遇到正面事件概率越高
- 社交圈越大，遇到贵人概率越高
- 连续创作，灵感事件概率增加
- 过度工作，健康/瓶颈事件概率增加

---

## 备注

- 随机事件每日检查一次
- 某些事件互斥（不会同时发生）
- 部分事件有前置条件
- 负面事件有时会带来意想不到的转机
