# 创作行为系统

本文档定义艺术家在游戏中的创作相关行为。

---

## 前期准备

### 1. 头脑风暴
| 属性 | 数值 |
|------|------|
| 行为ID | creative_brainstorm |
| 名称 | 头脑风暴 |
| 持续时间 | 1-2小时 |
| 体力消耗 | 10 |
| 灵感消耗 | 0 |
| 金钱消耗 | 0 |
| 效果 | 获得创作灵感，生成灵感笔记 |
| 触发条件 | 无 |

**产出**:
- 灵感点数: 3-8点
- 灵感笔记: 1-3条

**效率加成**:
| 状态 | 灵感获得 |
|------|----------|
| 心情好 | +30% |
| 听音乐 | +20% |
| 看展后 | +50% |
| 疲劳状态 | -50% |

---

### 2. 技法练习
| 属性 | 数值 |
|------|------|
| 行为ID | creative_practice |
| 名称 | 技法练习 |
| 持续时间 | 2-3小时 |
| 体力消耗 | 20 |
| 灵感消耗 | 5 |
| 金钱消耗 | 10-50 (材料) |
| 效果 | 提升指定技法熟练度 |
| 触发条件 | 拥有技法基础 |

**技法类型**:
| 类别 | 包含技法 | 练习消耗 |
|------|----------|----------|
| 绘画 | 素描/油画/水彩/版画 | 20体力 |
| 数码 | 3D建模/动画/交互/AI | 15体力 |
| 装置 | 焊接/组装/编程/电路 | 30体力 |
| 摄影 | 拍摄/后期/输出 | 15体力 |
| 影像 | 拍摄/剪辑/调色/特效 | 25体力 |

**熟练度提升**:
```
提升 = 基础值 * (1 + 导师指导加成 + 教材加成)
- 基础值: 5-15点
- 导师指导: +30%
- 优质教材: +20%
```

## 核心创作

### 3. 完成作品
| 属性 | 数值 |
|------|------|
| 行为ID | creative_complete |
| 名称 | 完成作品 |
| 持续时间 | 4-12小时 (视复杂度) |
| 体力消耗 | 30-60 |
| 灵感消耗 | 15-30 |
| 金钱消耗 | 50-2000 |
| 效果 | 创作新作品 |
| 触发条件 | 有创作想法 + 足够材料 |

**作品类型**:
| 类型 | 基础时长 | 体力消耗 | 灵感消耗 | 材料费 |
|------|----------|----------|----------|--------|
| 平面绘画 | 6h | 30 | 15 | 100 |
| 数码绘画 | 4h | 20 | 10 | 20 |
| 摄影作品 | 3h | 15 | 10 | 50 |
| 装置作品 | 12h | 60 | 30 | 1000 |
| 影像作品 | 8h | 40 | 25 | 500 |
| 交互作品 | 10h | 50 | 30 | 800 |

**作品质量公式**:
```
质量 = (技法熟练度 * 0.4 + 灵感值 * 0.3 + 体力 * 0.2 + 经验 * 0.1) * 随机(0.9-1.1)
```

**状态加成**:
| 状态 | 质量加成 |
|------|----------|
| 心流状态 | +20% |
| 完美工具 | +15% |
| 舒适环境 | +10% |
| 灵感爆发 | +30% |

|------|----------|----------|

- 过度修改可能导致作品"做坏"

## 创作状态

### 心流状态
进入条件:
- 连续创作2小时以上
- 体力>50
- 灵感>30
- 外界无干扰

效果:
- 创作效率+50%
- 作品质量+20%
- 灵感消耗-30%

### 创作瓶颈
触发条件:
- 连续创作5天以上
- 灵感<10
- 体力<20

效果:
- 创作效率-70%
- 作品质量-30%
- 强制休息

突破方法:
- 看展: 立即恢复
- 休息2天: 逐渐恢复
- 找人讨论: 随机恢复
