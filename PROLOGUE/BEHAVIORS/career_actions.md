本文档定义艺术家在游戏中的职业发展相关行为。

## 作品集管理

### 1. 投递作品集
| 属性 | 数值 |
|------|------|
| 行为ID | career_submit_portfolio |
| 名称 | 投递作品集 |
| 持续时间 | 2小时 |
| 体力消耗 | 20 |
| 金钱消耗 | 50-200 (打印/快递) |
| 经验产出 | +5 投递经验 |
| 触发条件 | 拥有完整作品集(≥5件作品) |

**投递目标**:
| 目标 | 成功率 | 反馈时间 | 成功奖励 |
|------|--------|----------|----------|
| 画廊代理 | 10-30% | 2-4周 | 展出机会 |
| 美术馆 | 5-20% | 1-3月 | 展览邀请 |
| 艺术节 | 20-40% | 1-2周 | 参展资格 |
| 驻留项目 | 15-35% | 3-6周 | 驻留机会 |
| 出版社 | 25-45% | 1-2周 | 出版机会 |

**成功率公式**:
```
成功率 = 基础成功率 + (作品质量 - 60) * 0.5% + (名声 / 10)
```

**优化策略**:
- 针对不同机构调整作品集风格
- 附上cover letter
- 通过人脉引荐 +20%成功率

## 学术发展

### 4. 申请学校
| 属性 | 数值 |
|------|------|
| 行为ID | career_apply_school |
| 名称 | 申请学校/深造 |
| 持续时间 | 准备2周，申请期1-3月 |
| 体力消耗 | 30 |
| 金钱消耗 | 100-500 (申请费) |
| 经验产出 | +10 申请经验 |
| 触发条件 | 满足学历要求 |

**学校类型**:
| 类型 | 学制 | 学费/年 | 竞争度 |
|------|------|---------|--------|
| 本科 | 4年 | 5-15万 | 中 |
| 硕士 | 2-3年 | 8-20万 | 高 |
| 博士 | 3-5年 | 3-10万 | 很高 |
| 进修班 | 1年 | 3-8万 | 低 |

**申请材料**:
- 作品集 (必须)
- 个人陈述 (必须)
- 推荐信 2封 (必须)
- 语言成绩 (部分必须)
- 学历证明 (必须)

**成功率公式**:
```
成功率 = 40% + (作品质量 - 65) * 1% + (GPA - 3.0) * 10% + 推荐信强度 * 10%
```

## 商业活动

### 6. 参加艺博会
| 属性 | 数值 |
|------|------|
| 行为ID | career_art_fair |
| 名称 | 参加艺博会 |
| 持续时间 | 4-7天 |
| 体力消耗 | 50 |
| 金钱消耗 | 2000-20000 (展位费) |
| 经验产出 | +30 销售经验 |
| 触发条件 | 被艺博会接受 |

**艺博会级别**:
| 级别 | 展位费 | 人流量 | 成交率 |
|------|--------|--------|--------|
| 初级市集 | 500-2000 | 500-2000/天 | 10-20% |
| 地区艺博 | 3000-8000 | 2000-5000/天 | 15-30% |
| 国际艺博 | 15000-50000 | 5000-20000/天 | 20-40% |

- 运输作品
- 布置展位
- 定价策略
- 宣传材料

**收益计算**:
```
收益 = 卖出金额 - 展位费 - 运输费 - 人工费
```
