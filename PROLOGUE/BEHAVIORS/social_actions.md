# 社交行为系统

本文档定义艺术家在游戏中的社交互动行为。

---

## 基础社交行为

### 1. 打招呼
| 属性 | 数值 |
|------|------|
| 行为ID | social_greet |
| 名称 | 打招呼 |
| 持续时间 | 5分钟 |
| 体力消耗 | 2 |
| 金钱消耗 | 0 |
| 效果 | 认识NPC，亲密度+5 |
| 冷却时间 | 无 |
| 触发条件 | 附近有NPC |

**对话选项**:
- 微笑点头 (亲密度+2)
- 热情问候 (亲密度+5，体力消耗+2)
- 假装没看见 (亲密度-5)

| 属性 | 数值 |
|------|------|

| 持续时间 | 30分钟 |
| 体力消耗 | 15 |

| 效果 | 发起合作项目 |
| 冷却时间 | 每月1次/每人 |

- 联合创作: 产出合作作品
- 项目协作: 获得项目经验
- 商业合作: 获得金钱

**成功率公式**:
```
成功率 = 30% + (亲密度 - 50) * 1% + (对方能力 - 50) * 0.5%
```

## 社交属性

### 亲和力
影响NPC对你的初始态度:
- 亲和力高: NPC更愿意见面
- 亲和力低: 可能被拒绝

### 名声
影响社交场合的表现:
- 名声高: 被更多人认识
- 名声低: 社交效果打折

### 人脉等级
衡量社交广度:
- 1级: 认识0-10人
- 2级: 认识11-30人
- 3级: 认识31-60人
- 4级: 认识61-100人
- 5级: 认识100人以上
