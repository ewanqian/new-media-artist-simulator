## 1. 布尔迪厄场域理论

### 1.1 核心概念

**场域（Field）**：
- 艺术场域是一个相对自主的社会空间
- 有自己的规则、资本和参与者
- 参与者通过资本争夺位置

**资本（Capital）**：

| 资本类型 | 描述 | 在本项目中的体现 |
|----------|------|------------------|
| 经济资本 | 金钱、财富 | 研究经费 |
| 文化资本 | 知识、教育、品味 | 理论深度 |
| 符号资本 | 声誉、认可 | 学术资本 |
| 社会资本 | 人脉、关系 | 人脉网络 |

### 1.2 场域二元对立

**有限生产场域 vs 大规模生产场域**：

```
有限生产场域（纯粹艺术）
    │
    ├── 评价标准：学术合法性、符号资本
    ├── 参与者：批评家、策展人、学者
    └── 目标：艺术自主性、先锋性

大规模生产场域（商业艺术）
    │
    ├── 评价标准：市场价值、经济资本
    ├── 参与者：画廊主、藏家、投资客
    └── 目标：商业回报、销售业绩
```

### 1.3 在本项目中的应用

玩家在两个场域之间移动，需要平衡：

```
学术路线 ← → 商业路线
   ↑           ↑
符号资本    经济资本
   ↑           ↑
策展人认可   作品销售
```

**场域位置影响**：
- 不同地点属于不同场域
- 不同派系偏向不同场域
- 玩家决策影响场域位置

## 2. 媒介理论

### 2.1 媒介即信息

麦克卢汉的核心观点：媒介本身比内容更重要。

在本项目中：
- 不同媒介决定不同的创作方式
- AI作为新媒介改变了创作流程
- 媒介选择影响艺术表达

### 2.2 媒介考古学

**核心问题**：旧媒介在新语境下有什么意义？

**研究对象**：
- CRT显示器
- 模拟信号
- 传统材料
- 旧技术

**创作方向**：
- 怀旧与反思
- 技术批判
- 语境重置

### 2.3 泛在计算

**核心概念**：计算无处不在

**影响**：
- 艺术创作的民主化
- 媒介边界的消解
- 艺术家身份的重定义

## 4. AI时代的艺术理论

### 4.1 创作主体性危机

**核心问题**：
- AI能否创作艺术？
- 谁是创作主体？
- 原创性如何定义？

### 4.2 理论回应

| 观点 | 代表人物 | 立场 |
|------|----------|------|
| AI创作论 | 部分AI研究者 | AI具有创造力 |
| 工具论 | 大部分艺术家 | AI是工具 |
| 协作论 | 新兴理论 | 人机协作 |
| 反思论 | 批判理论家 | AI引发自我反思 |

### 4.3 在本项目中的应用

玩家需要回答：
- 你如何看待AI创作？
- 你使用AI还是拒绝AI？
- 你如何定义自己的艺术主体性？
