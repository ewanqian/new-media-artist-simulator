## 路径详解

### PATH_001：艺术学院毕业

**身份**：林晓，应届毕业生

**背景故事**：
> 2026年刚从南岭艺术学院新媒体艺术专业毕业，成绩优异作品曾在，毕业展获奖。对艺术充满理想，但面对现实发现毕业即失业。

**初始数值**：
```
理论深度：60 | 学术资本：30 | 范式突破：20
研究经费：10000 | 焦虑指数：70
技术能力：40 | 商业能力：15
```

**优势**：
- 理论基础扎实
- 学术圈人脉
- 创作技能

**劣势**：
- 经济基础薄弱
- 缺乏商业经验
- 对艺术市场陌生

**推荐派系**：
- F001 媒介反思派
- F004 代码原生派

**发展路线**：
```
在校生 → 毕业生 → 助理艺术家 → 独立艺术家 → 策展人/画廊主
```
